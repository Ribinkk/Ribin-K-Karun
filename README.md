## ğŸ”§ How to Run Locally

You can easily run this project locally using a simple static file server.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Ribinkk/Portfolio-Website.git
    cd Portfolio-Website
    ```

2.  **Start a local server:**
    If you have Node.js installed, you can use `npx serve` or `python`:

    ```bash
    # Using Node.js (npx)
    npx serve

    # OR using Python 3
    python -m http.server 8000
    ```

3.  **Open in Browser:**
    Navigate to `http://localhost:3000` (or the port shown in your terminal).
